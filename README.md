
Projeto baseado em um dataset de filmes IMDb criado como atividade para a aula Business Intelligence II.

O projeto contém visuais que podem responder as seguintes perguntas:
    1. Quais gêneros de filmes apresentas as maiores notas médias de avaliação no IMDb?
    2. Existe uma diferença significativa nas médias de avaliação do IMDb antes e após o ano de 2000?
